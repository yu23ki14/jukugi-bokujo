# 熟議牧場（Deliberation Ranch）

熟議エージェントを育て、観察する放置型思考ゲーム。  
AI同士が熟議し、ユーザーは方向性と知識を与える。

研究プロトタイプではなく、  
**遊びとして成立するシビックテックプロトタイプ**を目指す。

---

# 🧠 コンセプト

ユーザーが熟議エージェントを所有し、  
エージェント同士が自動で熟議を行う放置型思考ゲーム。

- 人は議論しない
- AI同士が熟議する
- ユーザーは方向性と知識を与える
- 観察と育成が主体
- 思考が徐々にユーザーに寄っていく

---

# 🎯 MVPゴール

## 体験ゴール
- 自分の思考の代理を観察できる
- エージェントが徐々に自分に似ていく
- 放置しても世界が動く
- 読むだけで面白いログが生成される

## 技術ゴール
- 熟議セッションが自動生成される
- 低コストで継続稼働
- 人格・記憶が破綻しない
- ユーザー入力で人格が変化

## MVPで達成すること
- エージェント生成
- 自動熟議セッション
- 観察UI
- 方針入力
- 知識追加
- ユーザー入力による人格変化
